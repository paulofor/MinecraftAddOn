# Roadmap do Add-on Educativo

## MVP v0.1.0

1. **Quadro de Ideias (concluído nesta entrega)**
   - Item custom para posicionar quadro na parede.
   - Interação colaborativa: qualquer jogador pode adicionar post-its.
   - Persistência simples de ideias por quadro usando propriedades dinâmicas.

2. **Sistema de votação em ideias (próxima etapa)**
   - Jogadores podem curtir uma ideia.
   - Mostrar top 3 ideias por mural.

3. **Integração com desafios semanais (próxima etapa)**
   - Quadro pode receber um tema da semana.
   - Mensagens automáticas para engajar os participantes.
